# AstrBot 酒馆插件

当前目录存放的是 **AstrBot 酒馆大厅插件** 的实际主逻辑源码。

## 文件说明

- `main.py`：当前插件主文件

## 当前功能

- `酒馆启动`
- `剧本列表`
- `选择剧本`
- `加入酒馆`
- `退出酒馆`
- `开始酒馆`
- `开始游戏`（别名入口）
- `我的身份`
- `当前回合`
- `下一回合`
- `结束酒馆`
- `酒馆状态`
- `酒馆解散`
- `酒馆帮助`

## 当前行为特点

### 1. 人数规则

当前按剧本限制支持：

- 最少 3 人
- 最多 5 人

### 2. 身份发放

- `我的身份` 走私发
- 群里不会公开回落身份内容

### 3. 房间链接

- 房间页 / 旁观页链接只在创建酒馆时发一次
- 后续加入、开始、状态、回合推进等消息不再重复刷链接

### 4. 与 tavern-web 联动

插件通过以下配置与 `tavern-web` 交互：

- `TAVERN_WEB_ENABLED`
- `TAVERN_WEB_SYNC_URL`
- `TAVERN_WEB_PUBLIC_BASE_URL`
- `TAVERN_WEB_SYNC_TOKEN`
